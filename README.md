# EcoFit - Website for Environmental Protection Products Store

## 1. Overview
EcoFit is an e-commerce website designed to promote and sell eco-friendly and sustainable products.  
This project was developed as part of the **Business Web Development** course at the **University of Economics and Law (UEL)**.

The website aims to raise awareness about environmental protection by offering a variety of green products and an integrated GreenScore system to evaluate their eco-friendliness.

The system includes:
- A shopping page for customers to browse and purchase eco-friendly products.
- A GreenScore evaluation for each product.
- An admin dashboard for managing data and performance visualization.

---

## 2. Technologies and Tools
- Languages: HTML, CSS, JavaScript  
- Database: JSON (Local Storage)  
- IDE: Visual Studio Code  
- Library: Chart.js (for visualization)

## 3. How to Run the Project
No external installation or dependencies are required.

## 5. Main Features

### User Features
- Browse and view eco-friendly products.  
- Add items to the shopping cart and manage them (view, edit, delete).  
- Complete orders through the checkout process.  
- Choose between payment methods (Bank QR, MoMo, etc.).  
- View order summary and payment confirmation.  
- Track GreenScore to measure eco-friendliness.

### Admin Features
- Manage product list (view, add, edit, delete).  
- Manage customers and orders.  
- Analyze sales data and performance through charts.  
- Data visualization using Chart.js.

### Highlighted Features
- Modern and responsive UX/UI design.  
- GreenScore system to encourage sustainable behavior.  
- Interactive Admin Dashboard with real-time analytics.
